# Адаптивность сайта AI-Spetsnaz

## Общие принципы адаптивности

Сайт полностью адаптирован для отображения на всех устройствах:
- Мобильные телефоны (шириной до 768px)
- Планшеты (шириной от 768px до 1024px)
- Десктопы (шириной от 1024px и выше)

## Используемые breakpoints

Сайт использует следующие контрольные точки для адаптации:
- `sm`: 640px и выше
- `md`: 768px и выше
- `lg`: 1024px и выше
- `xl`: 1280px и выше
- `2xl`: 1536px и выше

## Адаптивные компоненты

### 1. Сеточные системы
- Используются классы Tailwind CSS: `grid-cols-1`, `sm:grid-cols-2`, `md:grid-cols-2`, `lg:grid-cols-3` и т.д.
- Для карточек применяются гибкие сетки с автоматическим перестроением

### 2. Изображения
- Все изображения имеют классы `w-full h-full object-contain` для корректного масштабирования
- Размеры изображений адаптируются под размер контейнера

### 3. Текст
- Размеры шрифтов изменяются в зависимости от размера экрана
- Используются относительные единицы измерения (rem, em)

### 4. Отступы
- Используются гибкие отступы с классами `gap`, `p`, `m`, `space-y`, `space-x`
- Отступы адаптируются под разные размеры экранов

### 5. Навигация
- Мобильная навигация скрывается на маленьких экранах и отображается через гамбургер-меню
- Десктопная навигация скрывается на маленьких экранах

## Особенности адаптации отдельных секций

### HeroSection
- Изменяется размер изображения мозга в зависимости от экрана
- Текст центрируется на мобильных устройствах
- Кнопки адаптируются под размер экрана

### ProblemSection
- Карточки перестраиваются из 2 колонок в 1 на мобильных устройствах
- Размеры иконок адаптируются под разные экраны

### ProductsSection
- Карточки перестраиваются из 3 колонок в 1 на мобильных устройствах
- Размеры изображений адаптируются

### TeamSection
- Карточки "Нанять" и "Арендовать" перестраиваются в одну колонку на мобильных устройствах
- Изображения команды адаптируются под размер экрана

### CTASection
- Форма перестраивается в одну колонку на мобильных устройствах
- Размеры полей ввода адаптируются

### SolutionsPage
- Карточки решений перестраиваются из 3 колонок в 1 на мобильных устройствах
- Размеры изображений адаптируются

## Оптимизация производительности

### Изображения
- Используются современные форматы изображений (WebP, SVG)
- Применяется ленивая загрузка изображений
- Изображения оптимизированы по размеру

### CSS
- Используется минификация CSS
- Применяются критические CSS для улучшения загрузки
- Используются современные свойства CSS (Grid, Flexbox)

### JavaScript
- Используется код-сплиттинг для уменьшения размера бандла
- Применяется ленивая загрузка компонентов
- Оптимизированы анимации с использованием requestAnimationFrame
