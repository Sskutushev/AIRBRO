# Настройка формы обратной связи с отправкой в Telegram

В проекте реализована форма обратной связи, которая собирает данные от пользователей и мгновенно пересылает их в указанный чат Telegram с помощью бота. Это позволяет оперативно получать заявки без необходимости настраивать почтовые серверы или сторонние сервисы.

## Принцип работы

1.  **Фронтенд (React)**: Пользователь заполняет и отправляет форму на сайте.
2.  **Бэкенд (Vercel Serverless Function)**: Сайт отправляет данные не напрямую в Telegram, а на специальную серверную функцию, развернутую на Vercel.
3.  **Серверная функция (Node.js)**: Эта функция принимает данные от сайта, форматирует их в сообщение и, используя секретный токен бота, отправляет его в ваш чат Telegram.

Такая архитектура гарантирует, что секретный токен вашего бота никогда не попадет в открытый доступ в код фронтенда.

## Пошаговая настройка

Для того чтобы форма отправляла заявки в **ваш** Telegram, нужно выполнить следующие шаги.

### Шаг 1: Создание Telegram-бота и получение токена

1.  Найдите в Telegram бота с именем `BotFather`.
2.  Отправьте ему команду `/newbot`.
3.  Следуйте инструкциям: придумайте имя для вашего бота (например, "Мой Сайт Бот") и его уникальный username (должен заканчиваться на `_bot`, например, `my_site_form_bot`).
4.  В ответ `BotFather` пришлет вам **токен** — длинную строку символов. **Сохраните его, это секретный ключ.**

### Шаг 2: Получение Chat ID

Чтобы бот знал, *куда* отправлять сообщения, ему нужен ID вашего чата.

1.  Найдите в поиске Telegram бота `@userinfobot`.
2.  Запустите его, и он сразу пришлет вам сообщение с вашим ID в строке `Id: 123456789`.
3.  **Сохраните это число.** Это ваш Chat ID.

### Шаг 3: Настройка для локальной разработки

Чтобы форма работала на вашем компьютере (локально).

1.  В корне проекта (`ai-spetsnaz-landing`) создайте файл с именем `.env`.
2.  Откройте его и добавьте две строки, подставив ваши значения из предыдущих шагов:

    ```
    TELEGRAM_BOT_TOKEN=ВАШ_ТОКЕН_ОТ_BOTFATHER
    TELEGRAM_CHAT_ID=ВАШ_ID_ЧАТА
    ```

3.  Для запуска локального сервера используйте команду `vercel dev`. Только она позволяет эмулировать серверные функции. Стандартная команда `npm run dev` не будет работать с формой.

### Шаг 4: Настройка для хостинга Vercel

Чтобы форма работала на реальном сайте после деплоя.

1.  Зайдите в ваш аккаунт на [Vercel](https://vercel.com/).
2.  Откройте ваш проект.
3.  Перейдите на вкладку **Settings -> Environment Variables**.
4.  Добавьте две переменные:
    -   **Key**: `TELEGRAM_BOT_TOKEN`, **Value**: `ВАШ_ТОКЕН_ОТ_BOTFATHER`
    -   **Key**: `TELEGRAM_CHAT_ID`, **Value**: `ВАШ_ID_ЧАТА`
5.  Убедитесь, что обе переменные доступны для всех окружений (Production, Preview, Development).
6.  Сохраните изменения. Vercel автоматически перезапустит проект с новыми переменными.

После выполнения этих шагов все заявки с формы будут приходить в ваш Telegram.
