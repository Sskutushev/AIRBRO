# Changelog

Все заметные изменения в проекте AIRBRO Business будут задокументированы в этом файле.

## [1.0.0] - 2025-11-05
### Добавлено
- Полная система аутентификации пользователей с регистрацией и авторизацией
- Система подписок и управления продуктами
- Поддержка светлой/темной темы с сохранением предпочтений
- Мультиязычность (RU/EN) с системой локализации
- Интеграция с Telegram Bot API
- Система оплаты с поддержкой различных методов (карта, Telegram Pay)
- 12 основных секций в соответствии с требованиями
- Система уведомлений и аналитики
- Личный кабинет пользователя
- Страницы оплаты и авторизации
- Контексты Auth и Subscription
- Сервисы подписок и платежей
- Все компоненты обновлены для работы с аутентифицированными пользователями

### Изменено
- Обновлена архитектура приложения для поддержки пользовательских сессий
- Улучшена система типизации и безопасности
- Обновлены все компоненты для работы с новыми сервисами
- Добавлена система локализации ко всем текстовым элементам

### Исправлено
- Ошибки типизации и импорта
- Проблемы с путями импортов
- Баги, связанные с аутентификацией
- Ошибки в работе с localStorage
